a generic uiml vocabulary for device - and modality independent user interfaces we present in this poster our work on a user interface markup language ( uiml ) vocabulary for the specification of device - and modality independent user interfaces . the work presented here is part of an application-oriented project . one of the results of the project is a prototype implementation of a generic platform for device independent multimodal mobile applications . the poster presents the requirements for a generic user interface description format and explains our approach on an integrated description of user interfaces for both graphical and voice modality . a basic overview of the vocabulary structure , its language elements and main features is presented .
