a quality framework for web site quality : user satisfaction and quality assurance web site developers need to use of standards and best practices to ensure that web sites are functional , accessible and interoperable . however many web sites fail to achieve such goals . this short paper describes how a web site quality assessment method ( e-qual ) might be used in conjunction with a quality assurance framework ( qa focus ) to provide a rounded view of web site quality that takes account of end user and developer perspectives .
